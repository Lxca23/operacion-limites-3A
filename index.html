¡Atención Cadete! Analiza las directrices teóricas y los ejemplos paso a paso antes de abordar los sistemas en vivo:

🟢 1. Límite de una Constante

Si el límite es de una constante, el resultado es siempre la misma constante ya que no hay letras que sustituir.
$$\lim_{x \to -3} 12 = 12$$

🟢 2. Límite de una Variable

Cuando calculamos el límite de una variable simple, la letra se reemplaza directamente por el valor numérico al cual tiende.
$$\lim_{x \to 7} x = 7$$

🟢 3. Multiplicación de Variable por Constante

Sustituimos el valor de la tendencia en la letra y efectuamos el producto final.
$$\lim_{x \to 4} (3x) = 3(4) = 12$$

🟢 4. Límite de una Suma

El límite de una suma se resuelve evaluando el valor de la tendencia y sumándole el término independiente.
$$\lim_{x \to 5} (x + 10) = 5 + 10 = 15$$

🟢 5. Límite de una Resta

Se evalúa la variable multiplicándola por su coeficiente correspondiente y restándole el valor numérico indicado.
$$\lim_{x \to 2} (4x - 3) = 4(2) - 3 = 8 - 3 = 5$$

🟢 6. Límite de una Potencia

Sustituimos el valor numérico de la tendencia dentro de la variable de la base y elevamos dicho número al exponente.
$$\lim_{x \to 3} (x^2) = 3^2 = 9$$

🟢 7. Límite de una Raíz (Radical Normal)

Evaluamos primero las operaciones internas debajo del radical y calculamos su raíz cuadrada exacta.
$$\lim_{x \to 4} \sqrt{x + 5} = \sqrt{4 + 5} = \sqrt{9} = 3$$

🟢 8. Límite Racional Normal

Sustituimos el valor en el numerador y denominador de forma independiente para obtener el cociente.
$$\lim_{x \to 2} \frac{x + 4}{x + 1} = \frac{2 + 4}{2 + 1} = \frac{6}{3} = 2$$

🔴 9. Indeterminación por Factorización

¡Alerta! Si da $\frac{0}{0}$, debes descomponer el numerador en binomios conjugados para cancelar el término indeterminado.
$$\lim_{x \to 3} \frac{x^2 - 9}{x - 3} = \frac{(x-3)(x+3)}{x-3} = x + 3 \implies 3 + 3 = 6$$

🔴 10. Racionalización de Radicales

Si produce una indeterminación $\frac{0}{0}$ con raíces, multiplicamos arriba y abajo por el conjugado del radical para eliminarla.
$$\lim_{x \to 4} \frac{\sqrt{x} - 2}{x - 4} \cdot \frac{\sqrt{x} + 2}{\sqrt{x} + 2} = \frac{x - 4}{(x-4)(\sqrt{x}+2)} = \frac{1}{\sqrt{x}+2} \implies \frac{1}{\sqrt{4}+2} = \frac{1}{4}$$
